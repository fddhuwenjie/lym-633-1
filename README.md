# 志愿者活动招募与工时认证系统

任务类型：0-1代码生成

业务领域：全栈Web应用

修改范围：跨模块多文件

任务难度：中等偏难

题面：

构建一个志愿者活动招募与工时认证系统，组织者发布活动、岗位、人数和服务时段，志愿者报名后由负责人确认到岗并登记服务时长。正常流程要支持报名、候补、签到、签退、工时审核、退回修正和证书生成。未满足岗位条件、同一时段报名多个活动、签退早于签到、负责人审核自己工时、活动结束后继续报名都要拦截。页面需要活动看板、岗位余量、个人服务档案和工时导出，导出包含活动、岗位、审核状态、证书编号和生成时间。重启后报名状态、候补顺序、签到记录、证书编号和导出历史都能复查。验收要检查岗位人数释放、工时退回后重新提交、证书撤销。
